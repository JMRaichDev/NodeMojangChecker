### Getting Started

**System Requirements**

* [Node.js][nodejs] v12

---

**Clone and Install Dependencies**

```console
> git clone https://github.com/JMRaichDev/NodeMojangChecker.git
> cd NodeMojangChecker
> npm install
```

---

**Launch Application**

```console
> npm start
```

---

**Build**

To build for your current platform.
```console
> npm install electron-packager -g
> npm electron-packager .
```
